# SubscriptionStatistics 订阅统计插件

## 功能介绍

本插件用于统计和分析订阅访问数据，支持通过 Telegram 机器人查看各种统计报告。

## 使用方法

### 基础命令

- `/sub` - 查看今日综合统计报告（各类排行显示 TOP 5）
- `/sub 7` - 查看最近7天综合统计报告
- `/sub 30` - 查看最近30天综合统计报告

### 专用排行命令

默认显示 TOP 20，支持自定义数量（1-100）：

#### 用户排行
- `/sub user` - 查看今日用户排行 TOP 20
- `/sub user 50` - 查看用户排行 TOP 50
- `/sub user 7 30` - 查看最近7天用户排行 TOP 30

#### IP访问排行
- `/sub ip` - 查看今日IP访问排行 TOP 20
- `/sub ip 30` - 查看IP访问排行 TOP 30
- `/sub ip 7 50` - 查看最近7天IP访问排行 TOP 50

#### 客户端(UA)排行
- `/sub ua` - 查看今日客户端使用排行 TOP 20
- `/sub ua 40` - 查看客户端使用排行 TOP 40
- `/sub ua 7 60` - 查看最近7天客户端使用排行 TOP 60

### 时间段说明

- 不写数字：今日数据
- `1`：昨日数据
- `2-30`：最近N天数据

### 数量限制

- 综合报告：各类排行固定显示 TOP 5
- 专用命令：默认 TOP 20，可自定义 1-100 个

## 数据统计说明

- **用户排行**：按订阅拉取次数排序
- **IP访问排行**：按IP地址访问次数排序
- **客户端排行**：按User-Agent使用次数排序（已合并相同客户端的不同版本）

## 权限要求

需要管理员或员工权限才能使用统计命令。
